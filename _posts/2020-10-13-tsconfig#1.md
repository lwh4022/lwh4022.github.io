# AllowJS
- JS 파일을 그대로 import 함, JS 모듈의 index.ts를 import함
- 기본 값은 false
```javascript
// @filename : card.js
export const defaultCardDeck = "Heart";

// @filename : index.ts
import { defaultCardDeck } from './card';

console.log(defaultCardDeck);
```
